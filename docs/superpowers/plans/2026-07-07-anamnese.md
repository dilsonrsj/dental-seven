# Anamnese Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or executing-plans.

**Goal:** Aba Anamnese na ficha do paciente com questionário estruturado v1, alertas críticos e persistência Supabase.

**Architecture:** Template fixo em TS (`template-v1`); uma linha `patient_anamnesis` por paciente com `responses jsonb`; server actions espelhando padrão de notas clínicas; aba nova na navegação do paciente.

**Spec:** `docs/superpowers/specs/2026-07-07-anamnese-design.md`  
**Guia:** `docs/superpowers/GUIA-MASTER.md` — v3.7 pós-beta

**Tech Stack:** Next.js 15, Supabase, Vitest, Tailwind, componentes UI existentes
